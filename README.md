# folder-tree


A basic folder creator app.

- Vue.
- ES6/ES7.
- Vuetify.

### Clone the repo

```bash
git clone https://github.com/sungkimcastro/folder-tree
```

### Install dependencies

```bash
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```
